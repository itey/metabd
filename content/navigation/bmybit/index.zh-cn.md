BmyBit 在2015 年底，由 Mario Stumpo 创立。BmyBit 开发了 PATS 全称为 Personal Automated Trading System 特有的交易系统，专用于设定交易策略和套利。这是一个基于云的运用预言分析，通过 API 互相连接数字交易市场的网页应用，便于对比不同市场间价格差进行套利。
